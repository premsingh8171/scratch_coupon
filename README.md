<h2>How to integrate the library in your app?</h2>
<b>Gradle dependency</b></br>

```gradle
dependencies {

	 	        implementation 'com.github.premsingh8171:scratch_coupon:1.0.0'
}
```

<b>Dependency</b></br>
```gradle
allprojects {
		repositories {
			
			maven { url 'https://jitpack.io' }
		}
	}
```
